# Simple E-commerce Store (Frontend Only)

A minimal, responsive e-commerce storefront built with HTML, CSS, and vanilla JavaScript. Features product listing, search, sorting, and a localStorage-backed cart with a slide-over drawer.

## Features
- Product grid with images, names, prices, categories
- Search and sort (popularity, price, name)
- Add to cart, modify quantities, remove items
- Cart drawer with total and demo checkout
- State persisted in localStorage
- Responsive design

## Getting Started
Open `index.html` directly in your browser. No server required.

If you prefer a local server (avoids some CORS issues on older browsers):
- With Python 3: `python -m http.server 8000`
- Then visit `http://localhost:8000`

## Notes
- This is a frontend-only demo. Checkout is simulated.
- Product images are from Unsplash and loaded via direct URLs.
